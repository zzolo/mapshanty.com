The xmppclient module provides a thin client for xmpp for the drupal platform
The thinclient is a custom written client called XWChat.

INSTALLATION
------------

Install and enable the xmppclient Drupal module as you would any Drupal module.

Configure the module at Administer > XMPPFramework > Client

DEPENDENCIES
------------
Module is written purely for Drupal 6.x core.
